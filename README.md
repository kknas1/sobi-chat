# 소비자분쟁해결 기준 계산기

> **⚠️ 이 프로젝트는 개인이 만든 비공식 도구입니다.**  
> 공정거래위원회 또는 한국소비자원과 무관하며, 법적 효력이 없습니다.

공정거래위원회 고시 **소비자분쟁해결기준(2025-14, 2025.12.18)** 별표 1~4를 바탕으로,  
소비자가 분쟁 내용을 채팅으로 입력하면 예상 환급·배상 기준 금액을 계산해주는 Node.js 앱입니다.

미약하지만 chrisryugj/korean-law-mcp 님이 만드신 법 관련 mcp를 보고 용기내서 만들었습니다.

나중에는 소비자분쟁해결 기준 문서를 마크다운 하고 DB화 해서 API로 호출, 검색가능하게 하는게 목표입니다.

그리고 장기적으론 공공기관의 내부 업무 시스템에 연동 할수 있는, 휴먼 에어를 줄일수 있는
내부 업무시스템을 깃에 배포하는게 제 목표 입니다.

예를 들어서...

출장 다녀오고 나서 출장비 받을려고 결의할때... 숙박 영수증, 톨비, 유류비 등등 증빙 자료를 빼먹거나
정산 잘못 하거나 해서 반려 많이 당했는데(저의 경우 ㅠㅠ)

내부 업무 시스템에 연계 할수 있는.. 신청 데이터를 읽어 오고 그걸 기반으로 증빙자료(이미지 등)에서 정확하게 데이터를 추출해서
AI가 한번 검증해주는 그런걸 만들려고 합니다.
(물론 공공기관은 보안이 중요하니까.. 내부 시스템에 넣을수 있는 AI 모델로..)

암튼 잘 되곘죠?

🔗 **라이브 데모**: [https://sobi-chat.onrender.com/)

암튼.. 실제 보상되는 금액은 사건마다 매우 다를수도 있는데 어쨋든 법의 기준대로 계산이 가능하게끔 만들어봤습니다.

법령 기준으로 룰베이스 기반으로 계산하고, 필요한 데이터는 open ai로 추출하는 그런 방식입니다.
API를 너무 많이 호출하면 제가 돈이 없어서 ㅠㅠ 안될수도 있습니다.

## 동작 방식

```
브라우저 → Render 서버 → OpenAI API
```

| 레이어 | 역할 |
|---|---|
| 프론트엔드 | 채팅 UI, 사용자 입력·출력 표시 |
| 서버 | 사건 상태 관리, 규칙 엔진 실행 |
| AI (OpenAI) | 품목·분쟁유형 추출, 후속 질문 생성 |
| 규칙 엔진 | 최종 금액 계산 (AI 미사용) |

`OPENAI_API_KEY`가 없으면 규칙 기반 추정으로만 동작합니다.

## 로컬 실행

### 1. 저장소 클론

```bash
git clone https://github.com/kknas1/consumer-dispute-calculator.git
cd consumer-dispute-calculator
```

### 2. 패키지 설치

```bash
npm install
```

### 3. 환경변수 설정

프로젝트 루트에 `.env` 파일을 생성합니다.

```env
OPENAI_API_KEY=sk-...        # 필수 (없으면 규칙 기반으로만 동작)
OPENAI_MODEL=gpt-4o-mini     # 선택 (기본값: gpt-4o-mini)
```

### 4. 서버 실행

```bash
npm start
```

브라우저에서 `http://127.0.0.1:3000` 접속

## Render 배포

1. [Render](https://render.com) 에서 `New +` → `Web Service` 선택
2. 이 저장소 연결
3. 아래와 같이 설정

| 항목 | 값 |
|---|---|
| Build Command | `npm install` |
| Start Command | `npm start` |

4. **Environment Variables** 추가

| 변수명 | 필수 여부 | 설명 |
|---|---|---|
| `OPENAI_API_KEY` | 필수 | OpenAI API 키 |
| `OPENAI_MODEL` | 선택 | 사용할 모델명 (예: `gpt-4o-mini`) |

> ⚠️ Render 무료 플랜은 일정 시간 미사용 시 서버가 슬립 상태로 전환됩니다.  
> 첫 요청 응답이 30초 이상 걸릴 수 있습니다.

### 법적 고지

- 이 프로젝트는 **개인이 제작한 비공식 도구**이며, 공정거래위원회 및 한국소비자원과 **무관**합니다.
- 소비자분쟁해결기준은 **분쟁 해결의 참고 기준**일 뿐이며, 실제 환급·배상 결과는 다를 수 있습니다.
- 이 서비스는 **법률 조언을 제공하지 않습니다.** 
- 계산 결과의 정확성을 보장하지 않으며, 결과 이용으로 인한 손해에 대해 책임을 지지 않습니다.

### 개인정보 안내

- 채팅창에 입력하신 내용은 **AI 분석을 위해 OpenAI 서버로 전송**됩니다.
- 분쟁 내용에 성명, 연락처, 계좌번호 등 **개인정보를 입력하지 마세요.**

### 기준 정보

- 적용 기준: **소비자분쟁해결기준 (공정거래위원회 고시 제2025-14호, 2025.12.18)**
